# 介绍 echo $?

# 在 Unix 和类 Unix 系统的 shell 环境中，`echo $?` 是一个常用的命令，用于检查上一个执行的命令的退出状态。
# 退出状态是一个数字值，它提供了关于命令执行结果的反馈。

# - 如果 `$?` 返回 `0`，这通常意味着上一个命令已成功执行，没有遇到任何错误。
# - 如果 `$?` 返回非 `0` 的值，这表示上一个命令执行失败或遇到了某种类型的错误。
#   不同的非零值可能对应不同的错误情况，具体取决于执行的命令和其内部错误处理机制。

# 例如，在尝试访问一个不存在的目录后，使用 `echo $?` 可以显示一个非零值，指示出现了错误。

# 这个功能在脚本编写和命令行操作中特别有用，因为它允许用户或程序根据前一个命令的成功或失败来采取进一步的操作。

# 使用方法：
# 1. 执行一个命令。
# 2. 紧接着输入 `echo $?` 并按回车。
# 3. 观察输出的数字值，以了解上一个命令的退出状态。

# 注意：`echo $?` 必须紧跟在你想检查其退出状态的命令之后，因为 `$?` 变量保存的是最近一次执行的命令的退出状态。
